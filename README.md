# Rhea Marie Grencio - Personal Portfolio

A modern, minimalist, and responsive personal portfolio built with React and Vite. It features a glassmorphism design, smooth scrolling animations, and dynamic blob backgrounds.

## 🚀 Features
- **React Components**: Modular component architecture for easy maintenance.
- **Modern UI**: Custom CSS (`index.css`) establishing a sleek dark mode.
- **Responsive**: Fully functional mobile navigation, scaling typography, and responsive grids.
- **Fast Build**: Propelled by Vite and easily deployable onto platforms like Vercel.

## 💻 Running the Project Locally

Follow these steps to clone and run the project strictly on your local machine.

### 1. Clone the repository
```bash
git clone https://github.com/Buenavistarm/MY-PORTFOLIO.git
cd MY-PORTFOLIO
```

### 2. Install Dependencies
Ensure you have Node.js installed, then install the packages:
```bash
npm install
```

### 3. Run the Development Server
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) to view the application locally.

### 4. Build for Production
To locally compile the application for a highly optimized production build:
```bash
npm run build
```
This generates the minimized static assets inside the `/dist` directory.

## 📷 Customizing Assets
All background images referenced in the Projects components can be placed directly inside the `public/images` or `src/assets` folder. Adjust the references in `src/components/Projects.jsx` accordingly.
